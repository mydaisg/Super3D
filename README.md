# Super3D

## Super 3D 分形算法研究

## 此算法的核心思想是将看似随机的号码序列通过"累计积分形转换"，揭示其可能存在的内在结构：

### 1、累计积分形转换：通过对原始号码序列进行累加，将点状的随机数据转换为平滑的趋势线。这种转换能有效过滤随机噪声，找出潜在的趋势性成分。

### 2、Hurst指数与分形维数：Hurst指数是衡量时间序列长期依赖性和趋势持续性的关键指标。通过R/S分析法计算，H值大于0.5表明序列具有持续性（趋势增强），小于0.5则具有反持续性（均值回归）。分形维数D=2-H，描述了序列的复杂度和不规则性。

### 3、趋势外推预测：基于近期累计积分形序列的线性趋势，预测下一期的累计和，再通过差分运算还原为原始号码的预测值。这种方法更关注序列的整体形态变化而非单个号码的出现。

## 💡 宽哥建议

### 1、参数调优：代码中的 window_size（滑动窗口大小）是关键参数。对于波动较大的序列，可尝试增大窗口值（如15-20）以平滑噪声；对于趋势变化的序列，可减小窗口值（如5-8）以提高灵敏度。

### 2、结合传统分析：可将分形分析结果与热冷号分析（热号指近期频繁出现的号码，冷号指长期未出现的号码）、奇偶比、大小比等传统方法结合

。例如，若分形分析预测某位置出大数（5-9），且该位置大数形态近期偏冷，则值得关注。
